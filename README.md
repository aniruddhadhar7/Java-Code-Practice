## 📖 About

> A well-organized collection of Java programming practice problems categorized by topic.

This repository is dedicated to improving Java programming skills through consistent practice. Each folder focuses on a specific concept, making it easier to learn, revise, and prepare for coding interviews and placement assessments.

---

# 📂 Topics Covered

## 🟠 Arrays
Programs covering array operations, searching, sorting, traversing, rotations, duplicate removal, and interview-based problems.

---

## 🔵 Basic Codes
Fundamental Java programs covering variables, loops, conditions, methods, operators, and core programming concepts.

---

## 🟢 NumberTypes
Mathematical programming problems including Prime Numbers, Armstrong Numbers, Perfect Numbers, Strong Numbers, Palindromes, Fibonacci Series, GCD, LCM, and more.

---

## 🟣 Patterns
Star patterns, number patterns, alphabet patterns, pyramids, hollow patterns, and nested loop exercises.

---

## 🔴 String
String manipulation programs including reversing strings, palindrome checking, character frequency, anagrams, duplicate characters, and more.
